# adminka-bat
A simple BAT script for Windows that allows administrators to obtain owner permissions and full access to a specified file or directory.
Adminka BAT

Простой BAT-скрипт для Windows, который позволяет администраторам получить права владельца и полный доступ к указанному файлу или каталогу.

Возможности
Смена владельца объекта через takeown
Выдача полного доступа группе Administrators через icacls
Поддержка файлов и папок
Рекурсивная обработка каталогов
Использование
Запустите adminka.bat от имени администратора.
Введите путь к файлу или папке.
Нажмите Enter.
Скрипт назначит владельца и выдаст полный доступ администраторам.
Пример
Path the way and Enter:
C:\Users\User\Documents\TestFolder
Требования
Windows 10 / 11
Права администратора
Внимание

Скрипт изменяет права доступа к файлам и папкам. Используйте его только для объектов, которыми вы имеете право управлять.
